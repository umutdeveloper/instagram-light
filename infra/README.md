# Infra

This folder contains infrastructure as code and deployment scripts for instagram-light.


## Development Setup

1. **Reopen in Container**
	- In VS Code, use the command palette to select "Remote-Containers: Reopen in Container" to start the dev environment.

2. **Start Services**
	- Run the following command to start Postgres and other services:
     
	  ```sh
	  docker-compose up -d
	  ```
